# cedar-game

大学模拟发展游戏 GitHub Pages 版本。

## 发布地址

启用 GitHub Pages 后，访问：

https://cedar4008.github.io/cedar-game/

## 上传方式

1. 在 GitHub 创建仓库：`cedar-game`
2. 上传本目录所有文件到仓库根目录
3. 打开仓库 Settings -> Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，目录选择 `/root`
6. 保存后等待 1-3 分钟

## 存档说明

本版本为纯静态网页，存档保存在当前浏览器 localStorage 中。
